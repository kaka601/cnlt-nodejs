Node.js Streams

Minh họa các loại Stream: Readable, Writable, Transform, Duplex

1. Writable Stream

Nhập dữ liệu vào form để nối vào đuôi file server (fs.createWriteStream).

Đã gửi yêu cầu!

2. Transform Stream

Gửi dữ liệu lên, server cho qua TransformFilter (UPPERCASE) rồi gửi về ngay lập tức.

3 & 4. Readable & Duplex Stream

Bấm nút In file để đọc nội dung file trên server, dữ liệu được truyền qua EchoDuplex trước khi trả về.
